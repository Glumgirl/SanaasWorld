    HTML Day 2026 - Manchester

    About the event

    HTML Day 2026 is an international event to celebrate the beauty and simplicity of HTML. Around the world, people gather to learn and write HTML together. For many people, myself included, HTML was the first programming language we learned, and it was to edit our tumblr, or if you're older, your MySpace or Geocities. You can do a lot with it, and that is the charm of HTML.

    This is the webpage for the Manchester edition

    Join us as we explore and experiment with HTML and create our own web pages~

    The event is open to everyone, whether or not you have any experience using HTML. If you're a beginner, we'll help you get started! Expect good company, html memes and a lot of snacks!
